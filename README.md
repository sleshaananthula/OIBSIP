<h2 style="font-size: 28px;"> Task List – Data Science</h2>


| Task No. | Project Title | Description |
|----------|---------------|--------------|
| 1 | **Iris Flower Classification** | Build a machine learning model to classify iris flowers (Setosa, Versicolor, Virginica) based on sepal and petal measurements. |
| 2 | **Unemployment Analysis with Python** | Analyze the unemployment rate in India, especially the impact of COVID-19, using data visualization and trend analysis. |
| 3 | **Car Price Prediction with Machine Learning** | Predict the selling price of a car based on features like brand value, horsepower, mileage, and other specifications. |
| 4 | **Email Spam Detection with Machine Learning** | Build a model that classifies emails as spam or non-spam (ham) using NLP techniques and classification algorithms. |
| 5 | **Sales Prediction using Python** | Predict product sales based on advertising spend across various platforms using regression techniques. |

<h2 style="font-size: 28px;"> How to Run</h2>


1. Clone the repository
   ```bash
   git clone https://github.com/<your-username>/OIBSIP.git
   cd OIBSIP
   ```
2. Navigate to the desired task folder
   ```bash
   cd Task1_IrisFlowerClassification
   ```
3. Install the required dependencies
   ```bash
   pip install -r requirements.txt
   ```
4. Open and run the notebook
   ```bash
   jupyter notebook iris_classification.ipynb
   ```
